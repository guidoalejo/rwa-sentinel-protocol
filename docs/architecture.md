# RWA Sentinel Protocol - Arquitectura

El proyecto RWA Sentinel Protocol utiliza un **Modelo 4** de liquidez, donde una IA actúa como oráculo legal: audita documentos legales (PDFs) y emite señales que permiten la tokenización segura de Activos del Mundo Real (RWA) en Blockchain. La arquitectura es híbrida: backend de IA en Python (ai-legal-core) y protocolo on-chain en Solidity (blockchain-protocol), coordinados por el oráculo legal.
